## 실행 방법

1. 레포지토리 클론

```bash
git clone -b leejin-yang --single-branch https://github.com/woowacourse/javascript-lotto.git
```

2. 의존성 설치

```bash
npm install
```

3. 콘솔 게임 실행

```bash
npm run start-step1
```

4. 콘솔 게임 실행

